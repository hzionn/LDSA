# LDSA

Learning Data Structures and Algorithms in C/C++.

## Prerequisites

- A C/C++ compiler, e.g. `gcc`, `clang`.
- to compile a C/C++ file, run

```bash
gcc -o <output_file> <source_file.c>
g++ -std=c++11 -o <output_file> <source_file.cpp>
```
